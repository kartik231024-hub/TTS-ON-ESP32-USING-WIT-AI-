# TTS-ON-ESP32-USING-WIT-AI-
📢 ESP32 Text-to-Speech (TTS) using Wit.ai  This project demonstrates how to implement Text-to-Speech (TTS) on an ESP32 using Wit.ai API and output audio through a MAX98357A I2S amplifier.  🚀 Features Convert text into speech using cloud API Real-time audio playback on ESP32 Works with low-cost hardware Clean and simple implementation
